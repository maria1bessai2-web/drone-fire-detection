# 🔥 Drone Fire Detection System

## 📌 Description
Conception et développement d’un drone intelligent dédié à la détection précoce des incendies, intégrant un modèle d’intelligence artificielle (YOLOv8), un système embarqué Raspberry Pi, la capture d’images, la géolocalisation GPS et la transmission en temps réel vers une interface de surveillance.
---

##  Fonctionnalités
- Détection automatique de feu avec IA
- Capture d’image lors de la détection
- Récupération des coordonnées GPS
- Envoi des données vers un serveur
- Alerte sonore et visuelle
- Interface web de surveillance en temps réel
- Historique des alertes

---

##  Technologies utilisées
- Python
- YOLOv8
- OpenCV
- Raspberry Pi
- Flask
- GPS M8N
- Communication serie

##  Architecture du système
1. Détection du feu via caméra
2. Traitement avec YOLOv8
3. Capture d’image
4. Récupération GPS
5. Envoi au serveur Flask
6. Affichage sur interface web

## Résultats
Le système permet de détecter les incendies en temps réel avec une précision élevée (~93%) et d’envoyer automatiquement des alertes avec image et localisation.

## Objectif
Améliorer la surveillance des zones à risque et réduire le temps d’intervention en cas d’incendie.
